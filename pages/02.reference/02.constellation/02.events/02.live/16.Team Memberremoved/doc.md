# team:{id}:memberRemoved

Sent when a team member leaves or invitee rejects their invite.

## Payload
|Type|Description|
|----|-----------|
|[User](/rest/index.html#User)|The member who left the team.|

## Example

{
  "level": 5,
  "social": {},
  "id": 696,

  "updatedAt": "2017-05-25T16:26:14.000Z",

}
